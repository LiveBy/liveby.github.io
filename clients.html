User API

The LiveBy User API providers data about users that register with LiveBy.

Example request 

https://localhost/v1/vendor/user?since=1480634634&upto=1480634664&brokerage=liveby&access_token=1GcfIcreeoOLXcXnQl6B%2Fqriggt79LIyGzMb5tc1YLU%3D
